# 按规则划分用户层级与奖金自动计算工具-creator-income-level-bonus
基于达人4周收益数据，自动完成用户分层、最终层级判定、奖金池计算与按层级分配。

## 项目功能
- 按固定收益区间自动划分8个用户层级
- 基于4周收益计算每人每周层级与最终层级
- 自动筛选获奖达人（层级≥2）
- 计算总奖金池（获奖总收益 × 3.5%）
- 按层级收益占比分发奖金，同层级人均奖金一致

## 使用场景
- 达人/创作者激励活动
- 用户分层运营
- 活动奖金自动核算
---

## 层级划分标准（共8级）
1. 0–300
2. 301–1000
3. 1001–3000
4. 3001–5000
5. 5001–8000
6. 8001–15000
7. 15001–30000
8. 30001–70000
---

## 最终层级判定逻辑
1. 统计4周中出现次数最多的层级
2. 若出现次数相同：
   → 计算**对应周收益总和**，取总和更大的层级
3. 无有效数据 → 默认层级1
---

## 奖金计算逻辑
1. 获奖条件：最终层级 ≥ 2
2. 总奖金池 = 所有获奖达人4周总收益 × 0.035
3. 按各层级**总收益占比**分配总奖金
4. 同层级内：人均奖金 = 层级总奖金 ÷ 该层级人数
---

## 使用方法
1. 准备 `达人收益.xlsx`，包含字段：
   - 第1周收益、第2周收益、第3周收益、第4周收益
2. 安装依赖：pip install pandas openpyxl
3. 运行脚本
4. 控制台输出各层级人均奖金
